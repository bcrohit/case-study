# case-study
This is a course work part of the Data Science degree program.



To compile LaTeX code install `LaTeX Workshop by James-Yu` and select Build Project by clicking `ctrl + shift + p` is copilot or vscode.

The LaTeX is split into three files 
- `presentation/shared_header.tex` contains header logos for all the slides
- `presentation/slides.tex` contain all the slides
- `presentation/week1_eda.tex` acts as the main file which pulls both other files and builds a PDF
